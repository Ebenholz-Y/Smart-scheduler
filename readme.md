# 智能排课系统

基于整数线性规划（ILP）建模的课程调度系统，解决课程、教师、教室三重资源冲突问题，实现高可用、低人工干预的自动化排课。

## 🔧 技术栈
- **后端**：Python 3.9+、Flask、PuLP（ILP 建模库）
- **求解器**：CBC（开源 MILP 求解器）
- **前端**：HTML/CSS/JavaScript（原生）
- **数据库**：MySQL
